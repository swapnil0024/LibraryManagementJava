# LibraryManagementJava
Library Management System - Java Console Application

A beginner-friendly backend-focused Java project that simulates a real-world Library Management System using Object-Oriented Programming concepts and Java Collections Framework.

Features:
Add and manage books
Search books by ID
Issue books to students
Return books with fine calculation
Track issued books
Menu-driven console application

Concepts Used:
Classes & Objects
Encapsulation
Constructors
ArrayList
HashMap
LocalDate & ChronoUnit
Object Relationships (HAS-A)
Service Layer Design
Scanner & Switch Case

Technologies:
Core Java
Java Collections Framework
Java Time API

Project Structure:
Book.java
Student.java
IssueRecord.java
FineCalculator.java
LibraryService.java
LibraryApp.java
Learning Outcome

This project helps in understanding:

Backend logic building
Real-world object modeling
Business logic implementation
Collection handling
Layered application structure

A great practice project for Java beginners preparing for backend development with JDBC, Hibernate, and Spring Boot later.
